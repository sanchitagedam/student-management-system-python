# student-management-system-python

This project is a simple Student Management System developed using Python and CSV file handling.

## Featues

- Add Studens 
- View Students
- Search Students
- Update Student
- Delete Student
- Prevent Duplicate IDs
- Count Total Students

## Technologies Used
- Python
- CSV File Handling

## How to Run

1. Download the project
2. Open terminal
3. Run

python student_management.py
